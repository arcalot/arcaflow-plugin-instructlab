# Python Plugin InstructLab

This plugin can run instructlab standalone, or as a part of a workflow.

## GPU support

GPU acceleration support is recommended for improving the performance of training.

To validate CUDA support, first run this command on the host system:
```commandline
nvidia-smi
```

Validate device availability with
```commandline
nvidia-ctk cdi list
```

You can then run `nvidia-smi` in the container to check compatibility in there.
```commandline
podman run -it --entrypoint /bin/bash --shm-size=5gb --device nvidia.com/gpu=all --security-opt label=disable <container image tag>
```

## Running the plugin

To run it outside a workflow with just podman, you can run it like this:
```commandline
 podman run -i --shm-size=5gb --volume /home/models:/home/models:z --volume /home/data:/home/data:z --device nvidia.com/gpu=all --security-opt label=disable quay.io/joconnel/arcaflow-plugin-instructlab --file - < inputs/cpu_offload.yaml
```
